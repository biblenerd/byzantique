The numbering of the Psalms in the Septuagint (LXX) differs from that of the Hebrew MT, and there is an additional Psalm in the LXX that is not present in the MT. Here is a mapping table:

| Greek LXX | Hebrew MT |
|-----------|-----------|
| 1–8 | 1–8 |
| 9 | 9–10 |
| 10–112 | 11–113 |
| 113 | 114–115 |
| 114 | 116:1–9 |
| 115 | 116:10–19 |
| 116–145 | 117–146 |
| 146 | 147:1–11 |
| 147 | 147:12–20 |
| 148–150 | 148–150 |
| 151 | *Not present* |
